# Mimari karar kayıtları (ADR)

Geri döndürmesi zor, güvenliği veya birden fazla modülü etkileyen kararlar burada numaralı olarak tutulur:
`0001-kisa-karar-adi.md`.

## Şablon

```markdown
# ADR-NNNN: Karar başlığı

- Durum: Önerildi | Kabul edildi | Geçersiz kılındı
- Tarih: YYYY-MM-DD
- Sahip: isim/rol

## Bağlam
Kararı gerektiren gerçekler, kısıtlar ve güvenlik etkisi.

## Seçenekler
Değerlendirilen seçenekler ve somut artı/eksileri.

## Karar
Seçilen yaklaşım ve nedenleri.

## Sonuçlar
Olumlu/olumsuz etkiler, migration, geri alma ve izleme.

## Kaynaklar
Güncel birincil kaynak bağlantıları.
```
